bdw_strsplit ()

string *
bdw_strsplit (conststring str,
              conststring delim);

Used to tokenize (or split) a given string by a given delimiter. Since this function uses strtok internally to tokenize str , it has the same limitations:

  • This function modify their first argument.

  • This function cannot be used on constant strings.

  • The identity of the delimiting byte is lost.

  • Uses a static buffer while parsing, so it's not thread safe.

For more information, see the man pages.

Parameters

str

String to be tokenized (split).

  

delim

String used as a delimiter for the tokenization.

  

Returns

The NULL-terminated array of strings result of the tokenization.

bdw_strstrip_leading ()

string
bdw_strstrip_leading (string str);

Removes leading whitespace from a string. This function doesn't allocate or reallocate any memory, it modifies str in place. Therefore, it cannot be used on statically allocated strings.

The pointer to str is returned to allow the nesting of functions.

Parameters
