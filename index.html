BdwLib
A general purpose library written in C.

Warning! This package is still under heavy development, expect bugs and issues pretty much everywhere.

BdwLib is a general purpose library written in C, it provides some utility functions for strings, files, threads, sockets and the most commonly used data structures like simple and doubly-linked Lists, Stacks and Queues.

BdwLib partially based on the structures and functions included in GLib.

It's planned to be available as an AUR package for ArchLinux and we are working on making it available for other Linux distributions such as Debian and Fedora.


Installation

Check the releases page for distribution-based installation options.

If there isn't a bdwlib package for your distribution, check the GitHub page for compiling and installation instructions.
